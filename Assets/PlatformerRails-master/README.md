# PlatformerRails
Character Controller for 2.5D Platformer on Unity  
made with Unity 2018.2.3f1

[日本語での紹介](https://friendsea.hateblo.jp/entry/2018/12/03/220409)

## Features

- Curved path
- Branch
- Changing gravity direction
- Deform meshes following paths
